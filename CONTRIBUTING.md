# Contributing

Thanks for taking the time to contribute.

## Before You Start

- Search existing issues before opening a new one.
- For anything larger than a small fix, open an issue first to align on scope.
- Keep pull requests focused and easy to review.

## Good Bug Reports

Please include:

- macOS version
- Better Clipboard version or commit hash
- whether the issue happens in the App Store build or a source build
- exact reproduction steps
- expected behavior
- actual behavior
- screenshots or a short recording if the bug is visual

## Scope Of This Public Repo

This repo is a public source mirror, not the full production app.

Source builds from this repo:

- stay on the free tier
- use demo clipboard events instead of the real production clipboard ingestion path
- do not include production purchase or release-only components

Please keep proposed changes within that public repo scope.
